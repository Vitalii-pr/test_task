### Test task
1. Clone repo
2. ```bash
    python3 -m venv ./venv
   ```
4. activate virtual environment
   ```bash
   source /venv/bin/activate```
5. ```bash
   pip install requirements.txt```
6. Run main.py file
7. Send a POST request to ```http://127.0.0.1:8000/summarize``` with a JSON body containing the text to be summarized.
